# Portifólio dos Projetos Integradores - Fatec

Trabalho de conclusão de curso na forma de um Portifólio de Aprendizagens, desenvolvido com base nos Projetos Integradores. Este trabalho é submetido à Faculdade de Tecnologia de São José dos Campos, como parte fundamental para cumprir os critérios estabelecidos na conquista do título de Tecnólogo em Banco de Dados.
